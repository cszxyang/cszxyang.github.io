关于我

  

名字是一个代号，我却有很多代号。朋友叫我临风、大雄、阿雄，家人叫我弟低、钊哥，还有网名麟渊、杨小胖……你更喜欢怎么称呼我呢？

目前主要从事的是 Java 后端开发，欢迎关注 GitHub账号。另外 旧博客 因为 2019 年重装操作系统丢失了所有 markdown 文件而停止维护，其中写的主要是一些 Computer Science 方面的学习笔记，个人想法方面的东西会少一些，自从看了 Steve Yegge 的 程序员的呐喊 开始觉得博客应该写多一些 “自己的、原创性的” 东西，而不是像目前国内很多所谓的博客那样，堆砌别人的内容（甚至还没注明出处），人云亦云。

自觉有一丝极客精神，向往工程师文化浓郁的工作环境，人生幸事之一是能有一份真心喜欢的工作，目前尚在寻觅。

可算喜欢读书，除了其中态度，更喜欢置身故事的感觉，那就好像作了一个天大的弊一般，因为人人都只有一条命，但身临其境的阅读却能让人好似体验了多段不同的人生。我赞同读书是一件私人的事情，但也愿意分享我正在体验的旅程，如果感兴趣可见我的 豆瓣档案。

可算热爱摄影，详情可见 这篇文章。
